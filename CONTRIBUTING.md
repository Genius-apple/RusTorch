# Contributing to RusTorch

Thank you for your interest in contributing to RusTorch! We welcome contributions from the community.

## Getting Started

1. Fork the repository.
2. Clone your fork: `git clone https://github.com/your-username/rustorch.git`
3. Create a feature branch: `git checkout -b feature/my-new-feature`
4. Make your changes and commit them.
5. Push to your fork: `git push origin feature/my-new-feature`
6. Submit a Pull Request.

## Development

- **Build**: `cargo build --workspace`
- **Test**: `cargo test --workspace`
- **Lint**: `cargo clippy --workspace`
- **Format**: `cargo fmt --all`

## Code Style

We follow the standard Rust coding conventions. Please ensure `cargo fmt` and `cargo clippy` pass before submitting.
